- 📂 Gerenciamento de arquivos via dashboard moderno.  
- 🚀 Tecnologias: **Next.js**, **React**, **TRPC**, **TailwindCSS**, **MySQL** e **Prisma**.

- 🌐 Salvar PDFs no banco de dados ou disponibilizá-los para download.
